Stock Price Analysis & Prediction

A data science project that analyzes historical Apple stock prices using Python, implements moving averages, visualizes trends, and forecasts future prices with the Prophet model.

Project Overview

This project explores Apple's stock price data from 2015 to 2024, applying various data analysis and machine learning techniques to understand historical trends and predict future price movements. By implementing moving averages, visualizing data patterns, and employing the Prophet forecasting model, this project demonstrates practical applications of data science in financial analysis.

Python Pandas Matplotlib Prophet yfinance scikit-learn

Key Features

  • Data Acquisition: Automated stock data retrieval using the yfinance API for Apple Inc. (AAPL) covering a 9-year period.
  • Technical Analysis: Implementation of 10-day and 50-day moving averages to identify trends and potential trading signals.
  • Data Visualization: Creation of informative charts showing price movements and technical indicators.
  • Time Series Forecasting: Application of Facebook's Prophet model to predict future stock prices.
  • Performance Evaluation: Calculation of Root Mean Square Error (RMSE) to assess prediction accuracy.

Implementation Details

Data Collection

The project begins by retrieving Apple stock data using the yfinance library, which provides a convenient way to download historical market data from Yahoo Finance:

import yfinance as yf import pandas as pd import os # Create 'data' directory if it doesn't exist os.makedirs('data', exist_ok=True) # Download data for Apple (AAPL) data = yf.download('AAPL', start='2015-01-01', end='2024-01-01') # Save to CSV inside the 'data' folder data.to_csv('data/AAPL.csv')

Technical Analysis with Moving Averages

Moving averages help smooth out price data to identify the direction of the trend. The project calculates 10-day and 50-day moving averages:

data['MA10'] = data['Close'].rolling(10).mean() data['MA50'] = data['Close'].rolling(50).mean() data = data.dropna()

Time Series Forecasting with Prophet

Facebook's Prophet is a powerful time series forecasting tool that handles seasonality and trends effectively:

from prophet import Prophet df = data.reset_index()[['Date', 'Close']] df.columns = ['ds', 'y'] model = Prophet() model.fit(df) future = model.make_future_dataframe(periods=30) forecast = model.predict(future) model.plot(forecast)

Model Evaluation

The project evaluates model performance using Root Mean Square Error (RMSE), which measures the differences between predicted and observed values:

import numpy as np from sklearn.metrics import mean_squared_error # Define actual and predicted values actual = df['Close'] predicted = df['Predicted_Close'] # Calculate RMSE rmse = np.sqrt(mean_squared_error(actual, predicted)) print("RMSE:", rmse)

Conclusions

This project demonstrates the application of data science techniques to financial market analysis. The combination of technical indicators like moving averages and advanced forecasting models like Prophet provides valuable insights for understanding stock price movements.

Key findings include:

  • Moving averages effectively identify underlying trends in Apple's stock price.
  • The Prophet model can capture both long-term trends and seasonal patterns in stock prices.
  • Performance metrics like RMSE help quantify prediction accuracy and provide a basis for model comparison.

Future improvements could include incorporating additional technical indicators, implementing sentiment analysis from news sources, and exploring more advanced machine learning models for prediction.
